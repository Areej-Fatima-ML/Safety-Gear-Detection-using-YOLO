# Safety-Gear-Detection-using-YOLO
# Project Overview 
Safety is a priority in industrial environments.This project automates the monitoring process by identifying
whether workers are wearing their mandatory safety gear using Computer Vision,

# Key Features
** End-to-End Pipeline: Data loading ,preprocessing,model training and inference all in one notebook.
** YOLO Architecture : Leverages the Ultralytics framework for high-speed object detection.
** Custom Training: Fine-Tuned on a safety gear dataset to detect PPE in real-time.

# Tools & Technologies
** Environment:Google Colab
** Language: Python
** Framework : YOLO(Ultralytics)
** Libraries:OpenCV,Matplotlib,PyTorch
